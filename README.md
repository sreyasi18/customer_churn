# Customer Churn Prediction & Dashboard

## Project Overview
This project predicts telecom customer churn using Machine Learning and visualizes customer behavior using Power BI.

## Technologies Used
- Python
- Pandas
- Scikit-learn
- SQL (phpMyAdmin/MySQL)
- Power BI

## Dataset
Telecom Customer Churn Dataset containing customer demographics, services, charges, and churn information.

## Project Workflow
1. Data Cleaning and Preprocessing
2. Exploratory Data Analysis (EDA)
3. Feature Engineering
4. Logistic Regression Model
5. Churn Prediction
6. SQL Analysis
7. Power BI Dashboard

## Model Performance
- Algorithm: Logistic Regression
- Accuracy: 90.97%

## Dashboard Insights
- Customers with month-to-month contracts have higher churn
- Fiber optic customers show higher churn tendency
- High monthly charges increase churn probability
